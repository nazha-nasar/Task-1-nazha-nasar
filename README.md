📊 Data Cleaning & Preparation Using Power Query
🚀 DecodeLabs Data Analytics Internship – Project 1

This repository contains Project 1: Data Cleaning and Preparation, completed as part of the Decode Labs Data Analytics Internship. The project focuses on transforming raw, unstructured data into a clean, consistent, and analysis-ready dataset using Power Query.

🎯 Objective
The primary goal of this project is to improve data quality and usability by performing structured data cleaning techniques. This ensures the dataset is reliable for downstream analysis and decision-making.

Key focus areas:
Data quality improvement
Handling missing and inconsistent data
Standardization of formats
Ensuring data integrity for analysis

🛠️ Tools & Technologies:
Power Query (Excel / Power BI)
Microsoft Excel
Data Preprocessing Techniques
Data Quality Management

🧹 Data Cleaning & Transformation Process

The following data preparation tasks were performed:
✔ Identified and handled missing values
✔ Replaced null values in CouponCode column with “No Coupon”
✔ Detected and validated duplicate records
✔ Standardized date formats and numeric fields
✔ Cleaned and formatted text data for consistency
✔ Ensured dataset uniformity for analytical readiness
